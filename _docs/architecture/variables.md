# Application environment variables

| **variable**             | **description**     | **default**         |
|--------------------------|---------------------|---------------------|
| HOST                     | URL host            | "0.0.0.0"           |
| PORT                     | PORT host           | "5000"              |
| DEBUG                    | DEBUG mode (on/off) | False               |
| APPLICATION_NAME         | Application name    | "ApplicationName"   |
